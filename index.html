Project on Console

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

A console is the combination of a monitor and keyboard. It is a rudimentary interface in which the monitor provides the output and the keyboard is used for input. Modern consoles often provide a graphical interface that can be used to control machines on a local network (LAN) or provide remote access to remote systems. We have to right click on the screen and inspect then go to the console section. We have to use console.log() to print anything on console.
